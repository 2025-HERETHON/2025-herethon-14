  ```
  └─ my_project
   ├─ my_project
   │  ├─ __init__.py
   │  ├─ asgi.py
   │  ├─ settings.py
   │  ├─ urls.py
   │  └─ wsgi.py
   ├─ myapp/
   │  ├─ __init__.py
   │  ├─ admin.py
   │  ├─ apps.py
   │  ├─ models.py
   │  ├─ tests.py
   │  └─ views.py
   └─ manage.py
  ```

  ```
  $ django-admin startproject 프로젝트명
  $ python manage.py runserver
  ```
